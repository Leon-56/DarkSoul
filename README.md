## DarkSoul复刻  　　
使用unity、C#开发的类魂系游戏，模型素材来自Asset Store，动画来自mixamo.com  

## 引擎工具　　
该项目使用 Unity2018.4 开发

## 游戏内容  
- 实现了简单的第三人称控制器，可以奔跑、跳跃、攻击、翻滚、被击、防御、盾反等等
- 实现战斗系统，玩家与敌人之间的攻击
- 运用Timeline完成盾反时间、宝箱开启
- UI(待填坑ing)
